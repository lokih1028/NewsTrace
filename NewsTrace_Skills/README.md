# NewsTrace Skills - README

## 📋 技能包概述

**NewsTrace** 是一个专业的金融新闻智能审计与回溯系统技能包,帮助开发者构建"基于数据的信任"的金融情报系统。

### 核心能力

- 🔍 **AI语义审计**: 识别新闻中的情绪化修饰、逻辑漏洞和翻译失真
- ⏱️ **时间胶囊追踪**: T+7自动化回测机制,验证新闻真实含金量
- 📊 **信源公信力评级**: 量化媒体可信度,生成红黑榜
- 🤖 **全流程自动化**: 从新闻接入到定时结算的闭环系统

## 💡 核心概念速览

### AI语义审计引擎

- 对比原文与译文,识别情绪化修饰
- 检测逻辑断层和煽动性表达
- 输出结构化审计结果 (JSON Schema约束)

### 时间胶囊机制

| 节点 | 时间 | 作用 |
|------|------|------|
| T0 | 新闻录入瞬间 | 基准价格 |
| T+N | 发布后1-6天 | 动态观察 |
| T+7 | 第7天 | 结案锁定 |

### 信源公信力评级

- 平均收益率 (40%)
- 辟谣率 (30%)
- 逻辑评分 (20%)
- 推荐准确率 (10%)

1. **Schema是契约**: 将LLM的感性判断约束为可执行字段
